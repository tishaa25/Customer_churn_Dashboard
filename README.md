# Customer Churn Analysis Dashboard

This project showcases a customer churn analysis dashboard created using Power BI to understand customer retention behavior.

## Tools Used
- Power BI
- Power Query

## Features
- Created KPIs such as total customers, churn count, and churn rate  
- Analyzed churn patterns based on contract types  
- Cleaned and transformed data using Power Query  
- Added slicers for interactive data filtering  

## Key Insights
- Higher churn observed in month-to-month contracts  
- Long-term contracts show better customer retention  
- Data visualization helps in identifying churn trends effectively
